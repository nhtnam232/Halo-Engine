# Halo – Cyber Access Engine

**Đồ án môn học:** Cơ sở Lập trình (Thực hành)

**Trường:** Đại học Khoa học Tự nhiên, ĐHQG-HCM (HCMUS)

---

## Giới thiệu
**Halo Engine** là một engine phân tích truy cập giúp tổ chức dữ liệu hiệu quả, tìm kiếm nhanh và hỗ trợ phát hiện hành vi đáng ngờ. 

Mục tiêu cốt lõi của đồ án là tối ưu hóa thuật toán và quản lý bộ nhớ nhằm đạt tốc độ thực thi nhanh nhất với mức tiêu thụ tài nguyên thấp nhất.
